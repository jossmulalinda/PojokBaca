# Project Rules — HMTI Unkhair Next.js

## Design Documentation Rule

Setiap kali ada perubahan signifikan pada proyek ini — baik penambahan halaman baru,
komponen baru, perubahan design system (warna, font, animasi), perubahan arsitektur,
atau penambahan library baru — **wajib update file `Design.md`** di root proyek.

### Yang wajib diupdate di Design.md:
- Komponen baru → tambahkan ke bagian **Struktur Komponen**
- Halaman/route baru → tambahkan ke bagian **Struktur Halaman (Routing)**
- Warna/token baru → tambahkan ke bagian **Palet Warna**
- Library baru → tambahkan ke bagian **Tech Stack** dan **Utilitas & Library**
- Keputusan desain baru → tambahkan ke bagian **Keputusan Desain**
- Animasi/CSS baru yang signifikan → tambahkan ke bagian **Animasi & Transisi**
